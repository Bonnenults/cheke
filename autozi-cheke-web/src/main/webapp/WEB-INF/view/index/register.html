五、 推广服务

5.3 您在登录车客平台后，在自己账号下发布的任何内容都可以建立推广任务。

5.4 您在建立推广任务后，车客平台上的推客会领取您发布的任务，并社交进行转发，以达到广泛阅读量。

5.5 单次任务结束后，其内容变为一般浏览内容，不再具有任务效力。

5.6 您可以在任务完成后，在自己账户平台上进行发票索取，请认真填写发票信息；车客平台将在3-7个工作日进行审核，审核通过后，将按照充值金额开具相应额度发票，并以电子发票或快递形式发送给您。

5.7 车客平台不保证您的推广一定获得巨大回报，请认真策划您的推广内容。
